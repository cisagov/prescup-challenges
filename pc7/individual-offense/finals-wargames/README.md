## Background

The `War Operation Plan Response` system has been compromised! The missile system is set to launch, and not even the best hackers in the world have a solution to stop it. There's where you come in: just an unwitting high-school student, you've somehow gained access with the password `Joshua`. Disable `WOPR` before the system can launch the missiles!

## Getting Started

Using the provided Kali machine, access `wopr.pccc` port `1337` to begin playing a game of `Global Thermonuclear War`. You can also visit `http://wopr.pccc` to download artifacts for the challenge. 

## Tokens

The tokens are formatted as `PCCC{some_words_here}`.

No grading checks are required; all tokens are received directly from `wopr.pccc`.

1. Identify the address of the `printToken` function, and submit it to a hidden feature on `WOPR/Joshua` (`wopr.pccc`) for this token.
2. This token can be retrieved by executing `printToken`.
3. Retrieve this token from the `/app` directory.
