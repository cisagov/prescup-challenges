# The Rabbit Hole

This challenge involves a chain of machines for you to break into, each one more difficult than the last.

**NICE Work Roles**

- [Exploitation Analyst](https://niccs.cisa.gov/tools/nice-framework/)

**NICE Tasks**

- [T1595](https://niccs.cisa.gov/tools/nice-framework/): Weaken Encryption
- [T1190](https://niccs.cisa.gov/tools/nice-framework/): Exploit Public-Facing Application

## Background

You have been given the task of breaking into several machines in a sequence. Your job is to find any other machines and break into them and gather any tokens you can find. Some of the challenges have inserted hints for you to read to determine what to do next.

## Getting Started

You have been given credentials to the `starter` machine. Each machine you gain access to will have very limited resources and tools for you to work with. Good Luck.
