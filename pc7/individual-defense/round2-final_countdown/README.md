# The Final Countdown

There is an active attack being carried out at a health center. Stop the attack without turning off critical services.

**NICE Work Roles**

- [Cyber Defense Incident Responder](https://niccs.cisa.gov/workforce-development/nice-framework/)

**NICE Tasks**


- [T0041](https://niccs.cisa.gov/workforce-development/nice-framework/): Mediate identified vulnerabilities
- [T0168](https://niccs.cisa.gov/workforce-development/nice-framework/): Contain and eradicate malicious activity
- [T0183](https://niccs.cisa.gov/workforce-development/nice-framework/): Analyze network and host-based logs to determine attack details

## Background

You are a cybersecurity professional working for a hospital. The hospital is actively getting attacked. You need to stop the attack and remediate the vulnerability that is allowing the attack to come through. You must not shut down any critical services used by the hospital. 

## Getting Started

Here are your tasks:

1. Remove the vulnerability that allows the attack to happen.
1. Remove all remnants of the attack.
    If you fail to fix the vulnerability in time, this task is harder.
1. Answer the following log analysis questions from the grader at `http://grader`:
    - What IP did the attacker reside?
    - What URL did the attacker use to exploit the vulnerability?
