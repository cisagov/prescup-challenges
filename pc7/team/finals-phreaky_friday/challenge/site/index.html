Overview

All artifacts are audio and voice-protocol traces. Token 2 leads into Token 3, however, the remaining tokens can be retrieved in any order. Use permitted tools (Wireshark, Audacity, sox, ffmpeg, Python).

Token 1 — The Whisper Trace

Artifacts: t1_whisper_trace.pcap + t1_calldata.wav

  1. Reverse the WHISPER protocol framing in the PCAP
  2. Extract payload fragments in order
  3. Decode the payload to retrieve an LSB audio fragment with ASCII token

Token 2 — CLAMSTREAM

Artifacts: t2_clamstream.bin

  1. Find CLAMSTREAM frame boundaries

Token 3 — BFSK Online

Artifacts: t3_snippet.wav

  1. Learn about the BFSK protocol
  2. Module power signatures to extract this token

Token 4 — HUSHRTP

Artifacts: t4_hushr.pcap + t4_hushr_clip.wav

  1. Extract handshake nonces
  2. Derive toy session key (KDF)
  3. Decrypt audio payload and demodulate base32

Token 5 — QUARTZ Bundle

Artifact: t5_quartz_bundle.zip

  1. Expose metadata secrets across formats
  2. Extract LSB plane in WAV
  3. Extract spectrogram glyphs and combine to final token
