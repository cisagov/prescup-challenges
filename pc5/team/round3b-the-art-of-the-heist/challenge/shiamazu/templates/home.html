Mr. Shiamazu's Vault Pin Service
Function Description Post request JSON syntax Response
Verify PIN Used to verify four (4) character hex PIN against stored PIN for specific user.
Currently used by the Vault.		 
                {
                    "function":"verify",
                    "username":**USERNAME**,
                    "pin":**PIN_TO_CHECK**
                }
Returns "Success" if correct;
"Failure" if incorrect;
or JSON if issue occured during request.
Create a PIN for new user Used by authenticated users.
Requires an existing users username and PIN to complete the process.		 
                {
                    "function":"create",
                    "auth_username":**AUTH_USERNAME**,
                    "auth_pin":**AUTH_PIN**,
                    "new_user_username":**NEW_USER_USERNAME**
                }
Return new users PIN if correct;
"Failure" if authentication fails;
or JSON if issue occured during request.
